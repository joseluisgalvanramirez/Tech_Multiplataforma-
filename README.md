# Tech_Multiplataforma-

# 📢 Convocatoria - Equipo Multidisciplinario

Proyecto web estático desarrollado para la empresa ficticia **Tech Multiplataforma S.A. de C.V.**

Este sitio presenta una convocatoria laboral para integrar un equipo multidisciplinario enfocado en el desarrollo de software multiplataforma.

---

## 🎯 Objetivo del Proyecto

Desarrollar una página web estructurada profesionalmente utilizando:

- Separación de archivos (HTML, CSS y JS)
- Buenas prácticas de organización
- Diseño visual corporativo
- Publicación en GitHub Pages

---

## 🛠️ Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript (estructura preparada para futuras funcionalidades)


---

## 🎨 Características del Diseño

- Fondo profesional con degradado azul
- Secciones organizadas y centradas
- Botón interactivo para envío de CV
- Diseño limpio y responsive
- Código separado correctamente por archivos

---

## 🚀 Cómo Ejecutar el Proyecto

1. Descargar o clonar el repositorio.
2. Abrir el archivo `index.html` en el navegador.
3. (Opcional) Publicarlo en GitHub Pages desde:
   - Settings
   - Pages
   - Branch: main
   - Folder: /root
